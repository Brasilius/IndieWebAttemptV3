## Overview

A personal portfolio website designed to host a record of both professional achievements, personal challenges and a general resume repository.

## Stack

- **Framework:** SvelteKit (Typescript)
- **Content:** mdsvex (Markdown + Svelte)
- **Styling:** Custom CSS variables with a matcha green palette

## Features

- File-based routing with TypeScript
- Markdown blog posts with frontmatter metadata
- Responsive design
- Light Mode + Dark Mode (tasteful color choices)
