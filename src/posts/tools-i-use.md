This is the tech I use on a daily basis for a wide variety of tasks across engineering and leisure.

## Hardware

**Main machine** - Framework 13 with AMD AI 340 (6 core 12 thread), 32 GBs of DDR5 5600, 2 TB (Samsung 990 Pro), 120 Hz Matte Screen, Fedora Linux! I use this machine for everything from CAD, to Web Dev.

**High Compute Power PC** - Custom PC I built - JonsboD32 Pro Case, Ryzen 7 7800X3D, RX 9070 XT, 32 GB DDR5 5600, 2 1TB (gen4) Drives, Fedora Linux/Windows 11 Pro, 5x 120mm Coolers, Peerless Assassin CPU Cooler.

(note mine doesn't have RBG!)

**Keyboard** - HHKB Studio, Snow Edition

![HHKB](/HHKB.jpg)


**Monitor** - Literally the cheapest 144 hz 27" 1440P panel that I could find. (also with USB-C connectivity)

## Software

**Editor** - VSCode on my main machine, but I do tend to use Micro/Nano in more resource constrained environments like Linux SBCs.

**Terminal** - Standard Fedora Terminal (Ptyxis)

**Browser** - Zen Browser (occasionally Brave incase something isnt working properly.)

**Notes** - ObsidianMD

**This site** - Built with [SvelteKit](https://kit.svelte.dev) and [mdsvex](https://mdsvex.pngwn.io).

**Additional Software** - WinBoat, I use this to "emulate" windows only software efficiently such as SolidWorks and the Microsoft Office Suite, without ever having to leave Linux!
